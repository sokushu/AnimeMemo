# 如何构建
- 项目开发环境：
    - Windows10
    - JDK1.8
    - Maven
    - VSCode 1.23.1
    - Mysql 5.7
- 如何编译
    - 首先要有Java运行环境和 Maven
    - 然后执行mav命令对pom.xml进行打包
    - 放到远程服务器上，运行jar包。
    - 服务器推荐使用 Nginx
